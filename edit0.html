Form W-7

The Internal Revenue Manual (IRM) provides necessary guidance for Form W-7 Edit and its Entry into RTS processing software. The ITIN IRM chapters are clearaly and logically organized into a Table of Contents, like in a book. You will find most necessary information using the Table of Contents. Most Form W-7 lines and each Real Time System (RTS) field is guided by several editing/processing rules. Every chapter/paragraph has its own title and a reference number (a nested number such as library/department/book/chapter/paragraph).

If you have trouble finding the necessary information following the Table of Contents, use the word search in SERP. (some IRM paragraphs and references are omitted due to their content not being suited for basic training)

Each highlighted field on Form W-7 bellow is guided by a set of edit rules and has its own IRM paragraph describing the rules. Each chapter/paragraph has its own reference number.

Hover over the stars ***** next to the highlighted fields to see the IRM reference with the link to the instructions.
